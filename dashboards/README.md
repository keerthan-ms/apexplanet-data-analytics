# Task 3: Data Visualization & Dashboarding 📊

For this phase of the internship at ApexPlanet Software Pvt Ltd, I translated raw customer acquisition data into a dynamic, executive-ready dashboard using **Looker Studio**. 

The dashboard tracks Total Revenue, Total Cost, and Average ROI across multiple marketing channels, highlighting that Email Marketing is the most capital-efficient acquisition stream.
